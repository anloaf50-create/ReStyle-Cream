Kebijakan Cookie

Terakhir diperbarui: 1 Januari 2025

ReStyle menggunakan cookie dan teknologi penyimpanan lokal untuk meningkatkan pengalaman Anda. Halaman ini menjelaskan apa itu cookie, jenis yang kami gunakan, dan cara mengelolanya.

Apa itu Cookie?

Cookie adalah file kecil yang disimpan di perangkat Anda ketika mengunjungi website. Cookie membantu website mengingat preferensi Anda dan meningkatkan pengalaman browsing.

Cookie yang Kami Gunakan

NamaTujuanDurasi
restyle-themeMenyimpan preferensi mode gelap/terangPermanen
restyle-cartMenyimpan isi keranjang belanjaSesi
restyle-userMenyimpan data login pengguna7 hari
restyle-ordersMenyimpan riwayat pesanan lokal30 hari

LocalStorage

Selain cookie, kami menggunakan localStorage browser untuk menyimpan data keranjang, pesanan, dan preferensi Anda secara lokal di perangkat Anda. Data ini tidak dikirim ke server kami secara otomatis.

Cara Mengelola Cookie

  • Anda dapat menghapus cookie melalui pengaturan browser kapan saja
  • Menonaktifkan cookie mungkin mempengaruhi fungsi website (misalnya: keranjang tidak tersimpan)
  • Panduan pengelolaan cookie tersedia di dokumentasi browser Anda

Cookie Pihak Ketiga

Kami menggunakan Google Fonts dan Font Awesome untuk tampilan website. Layanan ini mungkin menyimpan cookie mereka sendiri. Silakan periksa kebijakan privasi mereka untuk informasi lebih lanjut.

Dengan menggunakan ReStyle, Anda menyetujui penggunaan cookie sesuai kebijakan ini. Untuk pertanyaan: privacy@restyle.id
